# 轻阅读 - 飞牛应用

将 [轻阅读](https://github.com/autobcb/read) 轻阅读后端打包成飞牛 fnOS 应用。

## 使用

| 地址 | 说明 |
|------|------|
| `http://NAS-IP:15432/` | 阅读页面 |
| `http://NAS-IP:15432/admin` | 管理页面 |
